# Cores

As cores no nosso design system são importantes para identificação
do nosso produto além de ajudarem a criar experiências consistentes
através das nossas soluções.

<hr />
<br />

### Recomendações

-  Para facilitar a leitura e deixar a interface mais leve é
recomendável que as cores para tipografia, quando em backgrounds claros,
sejam tons de azul, e nunca cores “chapadas”.

-  Para melhor compreender a hierarquia das informações, é recomendável
a utilização de diferentes cores neutras representando hierarquias
distintas na escala tipográfica

-  Para melhor diferenciar vários tipos de dados e informações,
quando em mapas e tabelas, é recomendável a utilização de múltiplas cores

-  Para melhor comunicar o propósito principal da interface,
recomendamos a utilização da cor primária apenas uma vez por página,
ou o mínimo necessário.

-  Para facilitar a construção de modelos mentais de uso da interface,
recomendamos a utilização de cores de maneira consistente para a representação
de status, mensagens de feedback e convenções. 

Ex.: links azuis, mensagens de validação vermelhas, mensagens de sucesso azuis ou verdes.
