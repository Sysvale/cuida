# StatusItem

### Status items são componentes usados para mostrar um conjunto de informações sobre o estado em que determinada atividade se encontra, podendo exibir ações possíveis de acordo com a necessidade.
---

#### Recomendamos o uso de StatusItems quando:
- Você precisar exibir o estado de uma tarefa.
- Houverem ações que possam ser realizadas a partir destes estados.

#### Não recomendamos o uso de StatusItems quando:
- As ações a serem realizadas a partir dos estados sejam muito complexas.
