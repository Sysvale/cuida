# Checkbox

### Checkboxes permitem que os usuários selecionem um ou mais itens de um conjunto.
---
<br />

## Quando usar:
- Selecionar uma ou mais opções de uma lista.
- Apresentar uma lista contendo sub-seleções.
- Ativar ou desativar um item na área de trabalho.

<br />

## Quando não usar:
- Quando é necessário uma ação imediata. Prefira utilizar ToggleSwitch.
