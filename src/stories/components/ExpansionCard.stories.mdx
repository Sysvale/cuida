# ExpansionCard

### ExpansionCards são componentes utilizados para mostrar informações de forma resumida, com a possibilidade de ser expandido para detalhar essas informações.
---
<br />

## Recomendamos o uso de ExpansionCards quando:
- Forem necessários cards clicáveis.
- Se deseja mostrar informações resumidas.
- Se possui uma informação principal e um texto descritivo a ser colocado no card.

<br />

## Não recomendamos o uso de ExpansionCards quando:
- Houver um conjunto muito grande de informações a ser exibido.
