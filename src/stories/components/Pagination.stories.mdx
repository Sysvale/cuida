# Pagination

### Paginations são componentes utilizados para para dividir longos conjuntos de dados em diferentes páginas sem haver sobrecarrega do conteúdo na tela apresentada.
---
<br />

## Recomendamos o uso de Paginations quando:
- For necessário listar 20 dados ou mais.
- For necessário navegar para um conjunto de dados de forma paginada.

<br />

## Não Recomendamos o uso de Paginations quando:
- A quantidade de dados apresentados não é maior ou igual a 20.
- O conjunto de dados é apresentado em dispositivos moveis. Nesse caso, recomendamos o uso de rolagem infinita ([Lazy loading](https://developer.mozilla.org/en-US/docs/Web/Performance/Lazy_loading))
