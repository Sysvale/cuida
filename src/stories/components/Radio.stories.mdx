# Radio

### Radios são componentes utilizados para selecionar uma opção de um conjunto.
---
<br />

## Recomendamos o uso de Radios quando:
- For necessário apresentar duas ou mais opções que sejam simultaneamente exclusivas.

<br />

## Não recomendamos o uso de Radios quando:
- For necessário selecionar duas ou mais apções ao mesmo tempo.
- For necessário 4 ou mais opções. Nesse caso, é recomendado o uso do componente `Dropdown`.
