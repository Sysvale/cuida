# Scrollable

### Scrollable é um componente utilizado para o tornar o conteúdo envolvido "scrolável".
---
<br />

## Quando usar:
- Se quer mostrar muito conteúdo em um espaço pequeno ou pré-definido.

<br />

## Quando não usar:
- Houver pouco conteúdo.
- Já houver um outro scrollable.
