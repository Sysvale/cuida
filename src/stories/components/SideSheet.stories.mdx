# SideSheet

### SideSheets são componentes usados para mostrar conteúdo em uma estrutura de modal,
### mas que é apresentada em um dos lados da página.
---
<br />


## Recomendamos o uso de SideSheets quando:
- Se deseja colocar filtros que vão ser mostrandos ao clique de algum componente.

<br />

## Não recomendamos o uso de SideSheets quando:
- For necessário o uso de um modal convencional.
- For necessária uma solução de navegação.
- For necessário um form para registrar algo.
