# SideSheet

### SideSheets são componentes usados para mostrar conteúdo em uma estrutura de modal,
### mas que é apresentada em um dos lados da página.
---
<br />


## Recomendamos o uso de SideSheets quando:
- Você deseja colocar filtros que vão ser mostrandos ao clique de algum componente.

<br />

## Não recomendamos o uso de SideSheets quando:
- Você precisar de um modal convencional.
- Você precisar de uma solução de navegação.
- Você precisa de um form para registrar algo.
